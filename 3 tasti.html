Configura Firebase

Inserisci le credenziali del tuo progetto Firebase per abilitare la sincronizzazione in tempo reale tra le filiali di Cesena e Forlì.

Come ottenere le credenziali:
  1. Vai su console.firebase.google.com e crea un progetto
  2. Aggiungi un'app Web (icona </>)
  3. Vai in Realtime Database → Crea database (modalità test)
  4. Copia i valori da Impostazioni progetto → Le tue app
